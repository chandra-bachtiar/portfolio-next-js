The website is a weather information chatbot powered by Next.js 13 and the api.weatherapi.com API. It offers users a convenient way to retrieve weather updates for any location they desire. By interacting with the chatbot, users can inquire about current weather conditions, temperature, humidity, wind speed, and more. Leveraging the Next.js 13 framework, the website ensures a seamless and responsive user experience. With the integration of the api.weatherapi.com API, users can trust the accuracy and reliability of the weather data provided. Stay informed about the weather effortlessly with this intuitive and reliable weather chatbot website.

## Technologies Used
- Tailwind CSS
- DaisyUI
- NextJS 13
- NodeJS

## Install

Clone the repository
```bash
git clone https://github.com/chandra-bachtiar/WeatherChatBot.git
```

Install dependencies
```bash
npm install
```

Start the development server
```bash
npm run dev
```

if you want to build the project
```bash
npm run build
```
