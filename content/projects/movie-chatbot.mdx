The website is a movie information chatbot powered by Next.js 13 and the api.themoviedb.org API. It provides users with an interactive platform to obtain detailed information about any movie they are interested in. By engaging with the chatbot, users can inquire about movie titles, release dates, genres, cast members, ratings, and more. The Next.js 13 framework ensures a seamless and responsive user experience, while the integration of the api.themoviedb.org API guarantees access to a vast and up-to-date database of movies. Stay up to date with the latest movie releases and discover valuable insights about your favorite films with this user-friendly and informative movie chatbot website.

## Technologies Used
- Tailwind CSS
- DaisyUI
- NextJS 13
- NodeJS
- NPM

## Install

Clone the repository
```bash
git clone https://github.com/chandra-bachtiar/MovieChatBot.git
```

Install dependencies
```bash
npm install
```

Start the development server
```bash
npm run dev
```

if you want to build the project
```bash
npm run build
```
